💡 PRO TIPS:
═══════════════════════════════════════════════════════════════════════════════

1. GOOD LIGHTING IS KEY
   → Best results in natural daylight
   → Avoid harsh shadows
   → Keep hand clearly visible

2. HOLD GESTURES STEADY
   → Hold for 0.5+ seconds
   → Make clear, distinct motions
   → Full hand in frame

3. USE PHYSICAL DEVICE
   → Better performance than emulator
   → Faster detection
   → More realistic testing

4. MONITOR PERFORMANCE
   → Watch FPS counter
   → Adjust if needed
   → Close background apps if FPS < 20

5. EXTEND WITH EXAMPLES
   → 10 code examples included
   → Copy-paste ready
   → Just follow the pattern

📞 GETTING HELP:
═══════════════════════════════════════════════════════════════════════════════

IF CAMERA NOT WORKING:
  → Check app permissions
  → Restart app
  → Verify in device settings

IF GESTURES NOT DETECTED:
  → Improve lighting
  → Hold hand fully visible
  → Make clear, deliberate gestures
  → Hold gesture for 0.5+ seconds

IF PERFORMANCE IS LOW:
  → Close background apps
  → Use physical device (not emulator)
  → Lower device brightness
  → Reduce other device usage

IF MODEL WON'T LOAD:
  → Check internet connection
  → Free up storage space (need 100MB+)
  → Check firewall settings
  → Try again after restart

FOR CUSTOMIZATION:
  → Check EXAMPLES_AND_EXTENSIONS.ts

  → Refer to TensorFlow.js docs
