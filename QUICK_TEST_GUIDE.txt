COMMON ISSUES & SOLUTIONS
═══════════────────────────

ISSUE: App shows "Loading..." forever
SOLUTION:
  1. Check internet connection (needed first time)
  2. Wait another 5 seconds
  3. If still stuck, restart the terminal and run npm start again

ISSUE: QR code doesn't appear
SOLUTION:
  1. Wait 10+ seconds for Expo to start
  2. Make sure npm install completed successfully
  3. Try: npm start -c (clears cache)

ISSUE: App loads but camera shows black screen
SOLUTION:
  1. Grant camera permission when prompted
  2. Restart Expo Go app
  3. Scan QR code again

ISSUE: Only HI appears, no other gestures
SOLUTION:
  1. Wait another 10+ seconds (it cycles through)
  2. If still stuck, stop (Ctrl+C) and restart npm
  3. The fix has been applied - should work now

ISSUE: App freezes or becomes unresponsive
SOLUTION:
  1. Stop: Ctrl+C in terminal
  2. Run: npm start
  3. Scan QR code again
  4. The freeze bug should be fixed
