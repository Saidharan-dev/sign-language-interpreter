TROUBLESHOOTING QUICK REFERENCE
═══════════════════════════════════════════════════════════════════════════════

If npm start shows "Port 8081 in use":
  → This is NORMAL, it will use port 8082 automatically ✅

If QR code doesn't appear:
  → Wait 10-15 seconds more
  → Check internet connection
  → Kill terminal and retry

If app doesn't load on device:
  → Grant camera permission when prompted
  → Make sure Expo Go is latest version
  → Try pressing 'a' or 'i' in terminal instead of QR

If you see errors:
  → Check TROUBLESHOOTING_AND_FIXES.txt
  → Review PRE_RUNTIME_CHECKLIST.ts
  → Ensure npm install completed successfully
