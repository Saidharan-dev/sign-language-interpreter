HOW THE MOCK DETECTION WORKS
═══════════════════════════════════════════════════════════════════════════════

The app simulates hand detection by:

1. Creating 21 landmark points (same as real MediaPipe hand model)
   - 1 wrist point
   - 4 points per finger (CMC, MCP, PIP, TIP)

2. Positioning landmarks based on gesture type:
   - HI: All finger tips extended upward (low Y value)
   - PEACE: Only index and middle tips extended
   - FIST: All finger tips folded down (high Y value)
   - And so on...

3. Adding slight randomness to simulate natural hand variation

4. Feeding landmarks to recognizeGesture() function

5. Getting back actual analysis:
   - Which gesture was detected
   - Confidence level
   - Description
