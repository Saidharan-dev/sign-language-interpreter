🚀 HOW TO GET STARTED (3 SIMPLE STEPS)
═══════════════════════════════════════════════════════════════════════════════

STEP 1 - If not already done, install dependencies:
   $ npm install

STEP 2 - Start the development server:
   $ npm start

   You'll see:
   ┌──────────────────────────────────────┐
   │ Expo Development Server Started      │
   │                                      │
   │ ┌─────────────────────────────────┐  │
   │ │   QR Code                       │  │
   │ │   [████████████████]            │  │
   │ │                                 │  │
   │ │ Press 'a' for Android           │  │
   │ │ Press 'i' for iOS               │  │
   │ │ Press 'w' for web               │  │
   │ │                                 │  │
   │ └─────────────────────────────────┘  │
   └──────────────────────────────────────┘

STEP 3 - Run on your device:
   
   Option A - Android:
      • Download Expo Go from Play Store
      • Open Expo Go app
      • Tap "Scan QR Code"
      • Scan the QR code from terminal

   Option B - iOS:
      • Download Expo Go from App Store
      • Open Expo Go app
      • Point camera at QR code from terminal
      • Tap link that appears

   Option C - In Terminal:
      • Press 'a' for Android emulator
      • Press 'i' for iOS simulator
      • Press 'w' for web browser

After scanning/pressing:
   1. App loads (takes 5-10 seconds)
   2. "Loading model..." appears (wait 2-3 seconds)
   3. Camera feed appears
   4. Grant camera permission
   5. Ready for gesture detection!

🧠 HOW IT WORKS - TECHNICAL PIPELINE
═══════════════════════════════════════════════════════════════════════════════

1. CAPTURE (30 FPS)
   Camera → Video Frame
   │
2. DETECT (5ms)
   MediaPipe → 21 Hand Landmarks (3D coordinates)
   │
3. ANALYZE (10ms)
   Check Finger States → Calculate Distances/Angles
   │
4. SCORE (5ms)
   Generate Confidence for Each of 9 Gestures
   │
5. SMOOTH (5ms)
   Temporal Filtering → Remove False Positives
   │
6. DISPLAY
   Show Gesture + History + Metrics

Total Latency: ~50-100ms from hand motion to display

🆘 QUICK TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

PROBLEM: Camera not working
SOLUTION:
   1. Check app permissions (Settings → Apps → Camera)
   2. Grant camera permission
   3. Restart application
   4. Try different device

PROBLEM: Model won't load
SOLUTION:
   1. Check internet connection (first run needs download)
   2. Free up storage (need 100MB+)
   3. Check firewall isn't blocking CDN
   4. Restart device
   5. Try again in 5 minutes

PROBLEM: No gestures detected
SOLUTION:
   1. Improve lighting conditions
   2. Keep hand fully visible in frame
   3. Make gesture more clear and deliberate
   4. Hold gesture steady for 1+ seconds
   5. Check FPS (if <15, close background apps)

PROBLEM: App crashes on startup
SOLUTION:
   1. Close all background applications
   2. Free up device memory
   3. Reduce screen brightness
   4. Use physical device instead of emulator
   5. Clear app cache and restart

PROBLEM: Performance is slow (low FPS)
SOLUTION:
   1. Close background applications
   2. Lower device brightness
   3. Use physical device (not emulator)
   4. Reduce other usage (browsing, etc.)
   5. Wait for model to fully load
