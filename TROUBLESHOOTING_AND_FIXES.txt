Quick Troubleshooting
═══════════════════════════════════════════════════════════════════════════════

Q: "Port 8081 is in use" message appears
A: This is normal - Expo will automatically use port 8082 or higher. Continue.

Q: Blank screen after scanning QR code
A: Wait 10-15 seconds for the app to load, then check if it appears

Q: Camera permission popup doesn't show
A: Grant permission in device settings under the app options

Q: Gestures aren't changing
A: This is a demo - gestures appear randomly. Just wait a few seconds.

Q: Black screen stays
A: 
   1. Check internet connection (app needs it to download assets initially)
   2. Try restarting the terminal
   3. Kill npm and run again: npm start
   4. Clear Expo cache: expo start -c

Q: App crashes on startup
A: Kill terminal and try again:
   npm start
